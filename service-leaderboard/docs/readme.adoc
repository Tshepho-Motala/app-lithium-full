This document will describe the leaderboard framework, and how it all fits together and communicates to other services within lithium.

This service works by receiving messages from other services, which it uses to calculate percentages for promotion setups.
To participate in a promotion, a certain XP level is required. To learn more about XP, look at the link:../../service-xp/docs/readme.adoc[documentation] for XP.
