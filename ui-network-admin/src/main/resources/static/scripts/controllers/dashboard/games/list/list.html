Help

  • Below is a list of all the games that are available to your domain and any domains that you can manage.
  • You can edit some of the game information by clicking on the individual edit buttons for each game.
  • The game will only display in the customer-facing game list once you have enabled visibility by placing a tick on the checkbox under the column named visible for the game in question.
  • You can make a game invisible by removing the tick on the checkbox under the column named visible for the game in question.
  • You can enable a game on this page by placing a tick on the checkbox under the column named enable for the game in question.
  • You can disable a game on this page by removing the tick on the checkbox under the column named enable for the game in question.
